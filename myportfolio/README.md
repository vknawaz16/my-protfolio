# Premium Portfolio Website - Nawaz VK

A modern, premium personal portfolio website built with vanilla HTML5, CSS3, and JavaScript.

## Features

- Modern Glassmorphism + Cyber Professional UI
- Animated particle background
- Aurora gradient effects
- Smooth scroll reveal animations
- Mouse tracking spotlight effect
- Custom cursor glow
- Animated skill bars
- Project filtering system
- Certificate gallery with modal preview
- Responsive design for all devices
- Keyboard accessibility
- SEO-friendly structure

## Pages

1. **Home** (index.html) - Hero section with typing animation
2. **About** (about.html) - Introduction, timeline, strengths, stats
3. **Skills** (skills.html) - Categorized skills with animated bars
4. **Projects** (projects.html) - Project cards with filtering
5. **Coding Profiles** (coding-profiles.html) - Platform stats
6. **Certificates** (certificates.html) - Certificate gallery
7. **Resume** (resume.html) - Resume preview and download
8. **Contact** (contact.html) - Contact form and information

## How to View

Simply open `index.html` in your web browser. No build process required!

## Customization

### Change Profile Photo
Replace `assets/images/profile.jpg` with your photo. Then update the HTML files to use:
```html
<img src="assets/images/profile.jpg" alt="Nawaz VK" class="profile-image">
```

### Add New Projects
In `projects.html`, copy a project-card div and update the content.

### Add New Certificates
Place images in `assets/certificates/` and create certificate-card divs in `certificates.html`.

### Update Resume
Replace `assets/resume/resume.pdf` with your actual resume file.

### Edit Social Links
Update URLs in `coding-profiles.html` and `contact.html`.
